Margin, Border and Padding... What's the difference?

Margin:

The margin in CSS is the space around the outside of the element, changing it will change where the element is displayed on screen by creating a margin which pushes other elements away. It can be modified and changed to any desired value.

Border:

The border is the perimeter of the content being displayed. It can be modified to display different styles and sizes.

Padding:

Padding is used inside an element to create space. Padding is used to define the distance between the outer border of the element and the center of the element.
